Agora sobre o GitHub

O GitHub é a maior plataforma de hospedagem de código-fonte e colaboração do mundo, baseada na nuvem e focada em projetos que utilizam o sistema de controle de versão Git. Ele permite armazenar, gerenciar e compartilhar projetos de software, facilitando o trabalho em equipe (colaboração) através de recursos como pull requests, issues e revisão de código.

- Principal Diferença

Resumindo a diferença entre os dois, é que o Git é um sistema local de controle de versão de código que rastreia alterações no seu computador. GitHub é uma plataforma em nuvem que hospeda repositórios Git, permitindo colaboração e backup online. Essencialmente, o Git é a ferramenta, e o GitHub é o serviço que armazena essa ferramenta.
