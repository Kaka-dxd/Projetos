Entenda Finalmente a Diferença entre Git e GitHub

Git

O Git é um sistema de controle de versão distribuído. Em termos simples: ele guarda o histórico do seu código, permite voltar no tempo, trabalhar em equipe sem bagunça e testar mudanças com segurança. Ou seja, basicamente é uma ferramenta de versionamento.

Comandos Indispensáveis

📌 Git Init

- Cria repositórios na pasta atual

📌 Adicionar Arquivos

- git add nome-do-arquivo
- git add . (adiciona tudo)
